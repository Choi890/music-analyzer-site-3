# Auralis Studio

브라우저에서 오디오 파일을 분석하고 시각화하는 정적 웹사이트입니다. 업로드한 파일은 서버로 전송하지 않고 Web Audio API로 로컬 분석합니다.

## 실행

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.

## 포함된 분석

- 파일 정보, ID3 기본 태그, 길이, 샘플레이트, 채널, 추정 비트레이트
- 파형, 주파수 스펙트럼, 스펙트로그램, 라우드니스 곡선
- RMS/Peak dBFS, 다이내믹 레인지, 크레스트 팩터, 클리핑 비율
- BPM 추정, 키/모드 추정, 스테레오 폭, 중심 주파수, 롤오프
- 대역별 에너지, 밝기/따뜻함/공간감/텍스처 기반 감성 리포트

## 파일

- `index.html`: 화면 구조
- `styles.css`: 반응형 디자인과 오디오 반응 효과
- `script.js`: 업로드, 분석, 시각화 로직
